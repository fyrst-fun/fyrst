# Commit Conventions

This project follows the Conventional Commits specification.

## Format

```
<type>(<scope>): <description>
```

## Types

| Type | Description |
|------|-------------|
| feat | New feature |
| fix | Bug fix |
| docs | Documentation |
| style | Formatting |
| refactor | Code restructuring |
| test | Adding tests |
| chore | Maintenance |
| perf | Performance |
| ci | CI/CD changes |

## Scopes

| Scope | Description |
|-------|-------------|
| core | On-chain program |
| sdk | TypeScript SDK |
| ci | CI/CD workflows |
| readme | README updates |

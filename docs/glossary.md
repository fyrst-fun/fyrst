# Glossary

| Term | Definition |
|------|------------|
| Bonding Curve | Automated pricing function that determines token price based on supply |
| Collateral | SOL locked by deployer as security deposit |
| Deployer | Wallet address that launches a new token |
| Escrow | PDA-controlled vault holding deployer collateral |
| Graduation | Threshold where bonding curve freezes and liquidity migrates to DEX |
| PDA | Program Derived Address - deterministic account owned by the program |
| Pro-Rata | Proportional distribution based on individual contribution |
| Rug Pull | When a deployer abandons a token after extracting buyer funds |
| Safe Period | Time window during which collateral cannot be withdrawn |
| Slope | Rate at which token price increases per unit of supply |
